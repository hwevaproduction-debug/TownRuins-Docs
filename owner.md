<h3>Overview</h3>
<p>This dashboard is for <strong>platform owners</strong> at <strong>Hweva Tech Holdings</strong> — the people who own Town Ruins day to day, accept delivery, and run the platform after go-live. It is not a thin client overview for an external customer.</p>
<p>Product: <a href="https://app.townruins.com">https://app.townruins.com</a> · Support: <a href="mailto:sandbox@townruins.com">sandbox@townruins.com</a></p>
</div>

<div class="tr-dashboard-section">
<h3>Responsibilities</h3>
<ul>
<li>Own day-to-day platform operation after go-live</li>
<li>Use the admin panel as the main operating surface</li>
<li>Decide accounts, listings, bookings, tokens, and moderation</li>
<li>Know where support starts and stops (Support &amp; Warranty)</li>
</ul>
</div>
