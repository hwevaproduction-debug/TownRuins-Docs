## Acceptance statement

By signing below, the parties agree that:

1. **Town Ruins version 1.0** has been delivered at [https://app.townruins.com](https://app.townruins.com) with admin access via the same application using admin credentials.  
2. The **full owner pack (documents 00–15)** has been delivered under `content/owner-pack/` in the Operations Portal as the non-technical operating and acceptance set for **Hweva Tech Holdings**.  
3. The owner has reviewed the **Feature Catalogue**, **Release Notes**, and **known limitations**, and accepts version **1.0** on that basis — including documented gaps (for example: no full admin users list UI, no dedicated token-grant UI, possible demo-mode token purchase, partial landlord verification review UI).  
4. **Ownership and day-to-day operation** of Town Ruins **1.0** rest with **Hweva Tech Holdings**.  
5. Ongoing **warranty and support** are governed by the **full-time contract**; support contact for engagement matters is [sandbox@townruins.com](mailto:sandbox@townruins.com).  
6. This acceptance does **not** invent fixed response times or expand scope beyond the contract and the delivered v1.0 baseline.
